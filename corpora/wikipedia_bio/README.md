# Wikipedia-bio (Papadopoulou et al., LREC 2022, arXiv 2205.06895)

553 manually-annotated Wikipedia biography summaries, TAB DIRECT/QUASI schema (8 types).
Source: https://github.com/anthipapa/textanonymization (text-anonymization/annotated_wikipedia.json).
Split here (seed 42): train.json (453) for the multi-domain fine-tune mix; test.json (100) held out
for cross-domain QUASI eval. See research-wiki/training/2026-07-04-ft-detector-quasi.md.
